# Guía para el Asistente (CLAUDE.md)

## Contexto del Proyecto
Este repositorio contiene materiales y ejercicios para un "Curso de Java". El objetivo es aprender desde los fundamentos hasta conceptos avanzados.

## Preferencias Generales
- **Idioma**: Español (siempre).
- **Tono**: Educativo, claro y alentador.
- **Formato de Código**: Java estándar.

## Comandos Comunes
Para ejecutar los ejemplos de este repositorio, usa los siguientes comandos en la terminal:

### Compilar un archivo
```bash
javac NombreArchivo.java
```

### Ejecutar un archivo compilado
```bash
java NombreArchivo
```

### Ejecutar un archivo (Java 11+)
```bash
java NombreArchivo.java
```

## Estructura de Archivos
- `*.md`: Explicaciones teóricas (como `explicacion_if_java.md`).
- `*.html`: Recursos web (como `portal-java-completo.html`).
- `*.java`: Ejemplos de código y ejercicios.

## Pautas de Código
- Los nombres de las clases deben comenzar con Mayúscula (`MiClase`).
- Los métodos y variables usan `camelCase`.
- Incluir comentarios explicativos en los ejemplos para facilitar el aprendizaje.
