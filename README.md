# Curso Completo de Java ☕

¡Bienvenido al repositorio del Curso Completo de Java! Aquí encontrarás recursos teóricos, ejemplos prácticos y ejercicios para dominar el lenguaje de programación Java desde cero.

## 📂 Contenido del Curso

El material está organizado en diferentes módulos y archivos para facilitar tu aprendizaje.

### 🔹 Portal Principal
- **[Portal del Curso (HTML)](portal-java-completo.html)**: Una interfaz web interactiva con el temario completo, recursos y enlaces rápidos. Abre este archivo en tu navegador para comenzar.

### 🔹 Apuntes y Explicaciones
Aquí encontrarás documentos detallados sobre conceptos clave:

- **[Estructuras de Control: Sentencia IF y Operadores Lógicos](explicacion_if_java.md)**: Aprende a tomar decisiones en tu código y a utilizar operadores booleanos.

## 🚀 Requisitos Previos

Para aprovechar al máximo este curso, necesitarás tener instalado:

1.  **Java Development Kit (JDK)**: Recomendamos la versión 17 o 21 (LTS).
2.  **Un Editor de Código**: VS Code (recomendado), IntelliJ IDEA o Eclipse.

## 📖 Cómo Usar Este Repositorio

1.  Clona el repositorio o descarga los archivos en tu computadora.
2.  Abre el archivo `README.md` (este archivo) para tener una visión general.
3.  Navega por los archivos `.md` para leer las explicaciones teóricas.
4.  Ejecuta los ejemplos de código en tu IDE favorito para ver cómo funcionan.
5.  ¡Practica creando tus propios programas!
